隐私政策

最后更新：2026-06-16 · 生效日期：2026-06-16

简明摘要：我们仅收集您主动提供的数据（手机号、搜索查询），不卖您的数据，30 天后自动匿名化，账号注销 7 天内彻底删除。

1. 数据收集原则

我们遵循 最小必要 原则：

  • 不收集任何与服务无关的数据
  • 不卖任何用户数据给第三方
  • 不存搜索内容的长期记录（30 天后自动匿名化）
  • 加密存储密码（bcrypt）+ 传输（HTTPS）

2. 我们收集的数据

2.1 注册账号（必填）

  • 手机号：用于账号标识 + 验证
  • 密码：使用 bcrypt 加盐哈希存储，明文不存
  • 邮箱（可选）：用于密码重置

2.2 使用过程（自动）

  • 搜索查询：用于返回结果，30 天后自动匿名化（删除手机号关联）
  • IP 地址：用于速率限制 + 反滥用，7 天后自动删除
  • User-Agent：用于兼容性问题诊断

2.3 支付（购买会员时）

  • 订单号 + 金额 + 时间（支付宝返回）
  • 不收集银行卡 / 信用卡号（支付由支付宝处理）

3. 我们不收集的数据

  • ❌ 通讯录 / 短信 / 通话记录
  • ❌ 精确地理位置（仅 IP 推断的城市级）
  • ❌ 设备唯一标识符（IMEI/IDFA）
  • ❌ 相册 / 麦克风（仅 OCR 多模态搜索时，用户主动上传）

4. 数据使用范围

我们仅将数据用于：

  1. 提供搜索服务（必需）
  2. 防止滥用（速率限制 + 反爬虫，必需）
  3. 改进服务质量（聚合统计，匿名化后）
  4. 响应您的客服请求（您主动联系时）

5. 数据共享

我们不卖、不交换、不出租您的个人数据给任何第三方。

以下情况例外：

  • 法律要求：司法机关依法要求时
  • 您主动同意：您明确同意时

6. 第三方服务

我们使用以下第三方服务（仅接收返回结果，不发送您的个人数据）：

  • 搜索引擎：Bing CN / 搜狗 / 百度 / 头条 / 知乎 / 微信公众号 / GitHub / etc.
  • LLM 服务：GLM-4-Flash（智谱）— 仅发送搜索结果 + 您的查询
  • 支付：支付宝 — 支付时跳转
  • 监控：Prometheus + Grafana（自部署，公网 HTTPS）

7. 数据安全

  • 传输加密：全站 HTTPS（Let's Encrypt）
  • 存储加密：密码 bcrypt + 敏感字段 SHA-256
  • 访问控制：内部最小权限，外部不可访问数据库
  • 定期备份：每日异地备份，保留 30 天

8. 您的权利

您拥有以下权利（联系邮箱 stardust@token-star.cn 申请，7 天内响应）：

  • 查询：查看我们收集的您的所有数据
  • 更正：修改错误数据
  • 删除：注销账号（7 天内彻底删除）
  • 导出：导出您的搜索历史（JSON 格式）
  • 撤回同意：撤回注册时的数据使用同意

9. Cookie 与本地存储

  • localStorage：存储 star_token（HMAC token 12h 过期）+ 搜索历史
  • Cookie：不使用第三方追踪 Cookie
  • Service Worker：PWA 离线缓存（仅静态资源，不缓存搜索结果）

10. 儿童隐私

本服务不面向 14 岁以下儿童。如发现儿童注册，我们将主动删除账号。

11. 政策变更

如有重大变更，我们将：

  1. 提前 30 天 在 Web UI 公告
  2. 发送 邮件通知（如您已提供）
  3. 在变更生效前允许您导出 / 删除数据

12. 联系方式
